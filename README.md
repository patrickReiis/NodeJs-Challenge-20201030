## Como rodar o projeto:

## Criar database
```
$ psql postgres

postgres=# CREATE DATABASE food_challenge;
postgres=# \q
```

## Setup .env
Dentro da pasta root, crie um arquivo .env
O arquivo .env utiliza a seguinte estrutura:
```

DB_PASSWORD=EZFORCOiseverything
```
